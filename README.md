# Fract-ol
A fractal 2D explorer program

## HOW TO RUN IT ?
```
1. Clone the repo
2. make
3. ./fractol [fractal]
```

## LIST OF AVAILABLE FRACTAL :

1. Mandelbrot
2. Julia
3. Burning ship
